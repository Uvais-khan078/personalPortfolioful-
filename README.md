## Getting Started (Run Locally)

### 1) Clone the repo
```bash
git clone https://github.com/Uvais-khan078/personalPortfolioful-.git
cd personalPortfolioful-
```

### 2) Install dependencies
```bash
npm install
```

### 3) Start the development server
```bash
npm run dev
```

### 4) Build for production
```bash
npm run build
```

### 5) Preview production build
```bash
npm run preview
```

## How to Customize
Update these areas to make it truly yours:

- **Profile info:** name, role, intro text
